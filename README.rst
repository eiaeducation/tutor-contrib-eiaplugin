eiaplugin plugin for `Tutor <https://docs.tutor.edly.io>`__
###########################################################

eiaplugin plugin for Tutor


Installation
************

.. code-block:: bash

    pip install git+https://github.com/eiaeducation/tutor-contrib-eiaplugin

Usage
*****

.. code-block:: bash

    tutor plugins enable eiaplugin
